# Spring

## Spring 版本号

2.2.1.RELEASE

1. 第一个2 是主版本
2. 第二个2 是次版本 发布小的新特性 要保证兼容
3. 1 为增量版本 主要是bug修复
4. RELEASE 发布版本，里程碑版本 (RC, Alpha, Beta, GA = General Availability, SNAPSHOT比较稳定但不如GA 可用以了解新特性)
